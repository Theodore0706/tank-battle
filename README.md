# 坦克大战 (Tank Battle)

一个使用纯 HTML、CSS 和 JavaScript 实现的经典坦克大战小游戏。

## 在线试玩

直接在浏览器中打开 `tank-battle.html` 即可开始游戏。

## 操作说明

| 按键 | 功能 |
|------|------|
| `W` / `A` / `S` / `D` | 控制坦克上下左右移动 |
| `空格` / `J` | 发射炮弹 |
| `R` | 重新开始游戏 |

## 游戏规则

- 黄色坦克为玩家，红色坦克为敌人
- 消灭敌人获得 100 分
- 灰色方块为可破坏的墙壁
- 保护橙色基地，基地被击中则游戏结束
- 生命耗尽则游戏结束
- 清空所有敌人后进入下一关

## 本地运行

```bash
# 克隆仓库
git clone https://github.com/TheodoreXXX/tank-battle.git

# 进入目录
cd tank-battle

# 在浏览器中打开
open tank-battle.html
```

## 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3
